html table cheatsheet
Name Tag Description
Table <table> Defines a table
Table Head <thead> Defines a header cell in a table
Table Body <tbody> Groups the body content in a table
Table Row <tr> Defines a row in a table
Table Data <tbody> Defines a cell in a table
Table Footer <tfoot> Groups the footer content in a table
Caption <caption> Defines a table caption
Column Group <colgroup> Specifies a group of one or more columns in a table for formatting
Column Span <colspan> Defines how many columns a td element should span.
Row Span <rowspan> Defines how many columns a td element should span.
